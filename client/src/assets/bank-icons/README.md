# Ícones dos Bancos

Esta pasta contém os ícones dos bancos brasileiros para exibição na interface do sistema.

## Formato
- Formato: SVG ou PNG
- Tamanho recomendado: 24x24px
- Nomenclatura: código do banco (ex: 001.svg, 033.svg, 104.svg)

## Bancos Mapeados

- 033 - Banco Santander
- 104 - Caixa Econômica Federal
